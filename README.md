# HoroscopeBot

Телеграмм бот, который отправляет гороскоп выбранного занака зодиака.
Работать бот будет только тогда, когда включён компьютер и на нём запущена программа на Python. Если компьютер выключится, пропадёт интернет или вы отключите интерпретатор, то бот работать перестанет: запросы будут приходить, но никто на них не ответит.
